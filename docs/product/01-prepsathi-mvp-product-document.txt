                    1. EXECUTIVE SUMMARY
================================================================================

PrepSathi is a mobile-first exam preparation platform designed specifically 
for Nepali students preparing for Loksewa (Public Service Commission) exams.

MVP FOCUS:
----------
- ONE exam category from Loksewa (e.g., Administrative Officer/Kharidar)
- PHASE 1 ONLY: MCQ-based questions
  • General Knowledge
  • Nepal History
  • Current Affairs
  • Constitution & Law
  • Geography

WHY THIS MATTERS:
-----------------
• Thousands of students prepare for Loksewa with limited quality resources
• Traditional books are static - no feedback, no personalization
• AI can provide unlimited practice questions with instant feedback
• Mobile-first approach reaches students where they are

KEY DIFFERENTIATORS:
--------------------
✓ AI-powered question generation & feedback
✓ Personalized learning paths based on performance
✓ Instant feedback with detailed explanations
✓ Progress tracking & weak area identification
✓ Offline-capable for areas with poor connectivity
✓ Bilingual support (Nepali & English)

                    2. PRODUCT VISION & GOALS
================================================================================

VISION STATEMENT:
-----------------
"Make quality exam preparation accessible to every Loksewa aspirant in Nepal
through AI-powered personalized practice."

MVP GOALS:
----------
┌─────────────────────────────────────────────────────────────────────────────┐
│  GOAL                              │  SUCCESS METRIC                        │
├─────────────────────────────────────────────────────────────────────────────┤
│  User Acquisition                  │  1,000+ downloads in first month       │
│  User Engagement                   │  50%+ users practicing daily           │
│  Content Quality                   │  90%+ accuracy in questions            │
│  User Satisfaction                 │  4.0+ rating on app stores             │
│  Retention                         │  30%+ 7-day retention rate             │
└─────────────────────────────────────────────────────────────────────────────┘

CORE PRINCIPLES:
----------------
1. SIMPLICITY FIRST
   - Minimal screens, clear navigation
   - Focus on practice, not bells and whistles
   
2. QUALITY OVER QUANTITY
   - Better to have 500 verified questions than 5000 unverified
   - Every explanation must be educational
   
3. MOBILE-FIRST
   - Designed for smartphone usage
   - Works on low-end devices
   - Minimal data consumption

4. LEARN FROM MISTAKES
   - Every wrong answer is a learning opportunity
   - Detailed explanations, not just "correct/incorrect"

================================================================================
                    4. MVP SCOPE DEFINITION
================================================================================

IN SCOPE (MVP):
---------------
✅ One Loksewa exam category (Administrative Officer/Kharidar level)
✅ Phase 1 MCQ questions only
✅ Topics: General Knowledge, Nepal History, Current Affairs, 
   Constitution, Geography
✅ Phone number + OTP authentication
✅ Basic user profile
✅ Topic-wise practice
✅ Mock tests (timed)
✅ Progress tracking
✅ Question bookmarking
✅ Basic AI-generated explanations
✅ Bilingual support (English default, Nepali later)

OUT OF SCOPE (MVP):
-------------------
❌ Phase 2 (Written/Descriptive answers)
❌ Phase 3 (Interview preparation)
❌ Other Loksewa exam categories
❌ Other exam types (TSC, Banking, Police)
❌ Social features (leaderboards, friends)
❌ Paid subscription features
❌ Web application
❌ Video content/lectures
❌ Study notes/resources
❌ Chat with tutors

Priority Key: P0 = Must have, P1 = Should have, P2 = Nice to have


FEATURE: PHONE AUTHENTICATION
-----------------------------
• Nepal phone number validation (+977 97/98xxxxxxxx)
• OTP via SMS (6-digit code)
• 30-second resend cooldown
• Maximum 3 OTP attempts per session
• Session token stored securely (Expo SecureStore)
• Token refresh on expiry
• Logout clears all stored data

FEATURE: TOPIC PRACTICE
-----------------------
• Browse subjects → topics → sub-topics
• Filter: All / Unattempted / Incorrect / Bookmarked
• Difficulty levels: Easy / Medium / Hard
• Question count per topic shown
• Progress bar per topic
• Resume from where left off

FEATURE: QUIZ SESSION
---------------------
• Question with 4 options (A, B, C, D)
• Immediate feedback on selection
• Explanation after each question
• Bookmark during quiz
• Skip question option
• Progress indicator
• Exit with confirmation
• Timer (optional for practice, required for mock)

FEATURE: MOCK TEST
------------------
• Simulates real exam conditions
• 100 questions, 2-hour time limit
• No immediate feedback (review at end)
• Negative marking option
• Cannot pause timer
• Review all answers at end
• Detailed result analysis
• Compare with average scores

FEATURE: PROGRESS TRACKING
--------------------------
• Daily questions completed vs goal
• Streak counter (consecutive days)
• Overall accuracy percentage
• Subject-wise accuracy breakdown
• Topic-wise accuracy breakdown
• Weekly activity graph
• Time spent practicing
• Improvement trends

FEATURE: AI EXPLANATIONS
------------------------
• Generated for each question
• Includes: Why correct answer is right
• Includes: Why wrong answers are wrong
• Memory tips where applicable
• Related facts
• Source references (if from official)

================================================================================
                    9. AI INTEGRATION STRATEGY
================================================================================

AI USE CASES FOR MVP:
---------------------

1. QUESTION GENERATION
   ────────────────────
   • Generate new practice questions based on topics
   • Create variations of existing questions
   • Ensure Nepal-specific accuracy
   
   Approach:
   - Use seed questions as examples
   - Provide syllabus context
   - Validate generated questions before adding to DB
   - Initially: Human verification required
   - Later: Confidence score threshold

2. EXPLANATION GENERATION
   ───────────────────────
   • Generate detailed explanations for answers
   • Include memory tips
   • Add related facts for context
   
   Approach:
   - Generate on-demand or pre-generate in batches
   - Cache explanations for performance
   - Allow user feedback on explanation quality

3. WEAK AREA IDENTIFICATION
   ─────────────────────────
   • Analyze user performance patterns
   • Identify topics needing more practice
   • Suggest personalized practice sessions
   
   Approach:
   - Rule-based initially (accuracy < 60% = weak)
   - Later: ML-based pattern recognition

AI PROMPT TEMPLATES:
--------------------

Question Generation:
```
Generate 5 multiple choice questions about [TOPIC] for Loksewa exam preparation.

Context:
- Exam: Loksewa (Nepal Public Service Commission)
- Level: Non-Gazetted Second Class (Kharidar/Na.Su.)
- Subject: [SUBJECT]
- Topic: [TOPIC]
- Difficulty: [DIFFICULTY]

Requirements:
- Questions must be factually accurate about Nepal
- 4 options (A, B, C, D) for each question
- Only one correct answer
- Options should be plausible (no obviously wrong answers)
- Include the year in BS (Bikram Sambat) where applicable
- Reference official sources where possible

Similar questions for reference:
[SEED_QUESTIONS]

Output format:
{
  "questions": [
    {
      "question": "...",
      "options": { "a": "...", "b": "...", "c": "...", "d": "..." },
      "correctAnswer": "a",
      "explanation": "...",
      "difficulty": "medium"
    }
  ]
}
```

Explanation Generation:
```
Generate a detailed explanation for this Loksewa exam question.

Question: [QUESTION]
Options:
A) [OPTION_A]
B) [OPTION_B]
C) [OPTION_C]
D) [OPTION_D]

Correct Answer: [CORRECT_ANSWER]

Generate an explanation that:
1. Explains why the correct answer is right
2. Briefly explains why other options are wrong
```

PERFORMANCE CONSIDERATIONS:
---------------------------
• Lazy load questions (paginate)
• Cache frequently accessed data
• Optimize images (use WebP)
• Minimize bundle size
• Use FlatList for long lists
• Debounce search inputs
• Prefetch next questions during quiz

OFFLINE SUPPORT:
----------------
• Download question sets for offline practice
• Queue answers for sync when online
• Show last synced timestamp
• Graceful degradation for features needing internet

SECURITY:
---------
• Store tokens in SecureStore (not AsyncStorage)
• Implement token refresh mechanism
• Rate limit API endpoints
• Validate all inputs server-side
• Encrypt sensitive data in transit (HTTPS)
